You will have to download both of these for this to work.
If you already have project title, you will have to replace your version of project title with my version to make the navbar work. 
Installation:
Navbar- just put it in your patches folder in koreader
My version of project title- unzip it and put the plugin into koreader in the plugin folder. Replace project title if you already have it. 
Side note: Each time you change the size of the navbar, you will need to restart koreader to allow the UI to reshape itself and prevent overlapping
